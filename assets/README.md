## Alternative: Generate DAG with Code

If you want to generate Figure 3 programmatically, you can use:

```python
import graphviz

dag = graphviz.Digraph('DAG', format='png')
dag.attr(rankdir='LR')

# Add nodes
dag.node('Deaths', 'Deaths from\nNatural Disasters')
dag.node('Tourists', 'Tourist\nArrivals')
dag.node('GDP', 'GDP per capita')
dag.node('PopDens', 'Population\nDensity')
dag.node('Prep', 'Preparedness\n(WRI)')
dag.node('Geo', 'Geography')
dag.node('Climate', 'Climate')

# Add edges
dag.edge('Deaths', 'Tourists', label='β')
dag.edge('GDP', 'Tourists', label='γ₁')
dag.edge('GDP', 'Deaths')
dag.edge('PopDens', 'Tourists', label='γ₂')
dag.edge('PopDens', 'Deaths')
dag.edge('Prep', 'Tourists', label='γ₃')
dag.edge('Prep', 'Deaths')
dag.edge('Geo', 'Deaths')
dag.edge('Geo', 'Tourists')
dag.edge('Climate', 'Deaths')
dag.edge('Climate', 'Tourists')

dag.render('assets/figure3_dag', view=False)
```
